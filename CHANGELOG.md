# Changelog

## [1.0.0] - 2026-04-15

### 🎉 Initial Release

- **Sidebar Chat UI** — VS Code 사이드바에서 로컬 AI와 채팅
- **Ollama Integration** — 로컬 Ollama 서버와 HTTP API 연동
- **Real-time Streaming** — SSE 기반 토큰 단위 실시간 응답
- **Markdown Rendering** — 코드 블록, 헤더, 리스트, 인라인 코드 지원
- **Code Context** — 현재 파일/선택 영역 자동 컨텍스트 주입
- **Code Actions** — 응답 코드 블록 복사/에디터 삽입
- **Model Switching** — 상태바 클릭으로 모델 핫스왑
- **Keyboard Shortcut** — `Ctrl+Shift+L` 선택 코드 질문
- **Custom Modelfile** — E4B 페르소나 Ollama Modelfile 포함
